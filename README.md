# Workout Tracker

A simple full stack web application for tracking workouts and exercises.

Users can create workouts, add exercises to each workout and manage their workout history.
---

## Features

- Create, view, update and delete workouts
- Add and remove exercises associated with a workout
- Frontend interactions using JavaScript (Fetch API)
- REST-style backend API built with Flask
- Persistent data storage using SQLite

---

## Tech Stack

**Backend**
- Python
- Flask
- SQLite

**Frontend**
- HTML
- CSS
- JavaScript (Fetch API)

**Tools**
- Git & GitHub
- Virtual environments (`venv`)

---

## How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/windwalker46/workout-tracker.git
cd workout-tracker

2. Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Run the app
python app.py

5. Open from your browser
http://127.0.0.1:5000
```
